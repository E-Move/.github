# E-MOVE

Um projeto de e-commerce utilizando tecnologias modernas.

## Tecnologias Utilizadas

### Front-end:

- Vue.js;
- Prettier;
- ESlint;
- Pinia;
- Axios;
- TailwindCSS;
- Vuetify.

### Back-end:

- Node.js;
- Express
- TypeScript;
- Redis;
- Prisma;
- PostgreSQL.
